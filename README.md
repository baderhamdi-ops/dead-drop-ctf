
# Dead Drop — CTF Challenge

**Category:** Network Forensics  
**Difficulty:** Medium
**Flag format:** `securinetsenit{...}`

## Description
A suspect was arrested, but their C2 channel is still live.
We captured some traffic off the wire — somewhere in here
they dead-dropped something important. Find it.
